The garden follows the **golden angle (137.5Â°)**, the same pattern found in sunflowers, galaxies, and shells. Each plant is positioned mathematically, yet feels organic. Order within chaos. Structure within nature.

### The Lighthouse

The lighthouse is both anchor and beacon. Click it to reveal the core mission. Its light sweeps across the garden, pulsing brighter in darknessâ€”always guiding, always present.

### The Day/Night Cycle
